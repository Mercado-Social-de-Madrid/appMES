# :material-view-dashboard-edit-outline: Panel de gestión

Como se explicaba al inicio. Esta aplicación, además de la parte móvil, tiene una parte en la web tanto pública como privada.
En este área privada podremos gestionar nuestra información y realizar algunas operaciones como entidad o consumidora
(esta última depende de cada mercado). 

Se accede a través de esta dirección web:

#### [mercadosocial.app/login](https://mercadosocial.app/login/){ target=blank }

Aquí introducimos nuestros datos de acceso.


/// admonition | ¿No tienes datos de acceso?
    type: warning
Si no tienes tu email y contraseña para acceder, pueden pasar varias cosas:

- Todavía no te ha hecho socia del mercado social: Para darte de alta tienes que dirigirte al mercado territorial correspondiente
y seguir su proceso de solicitud y acogida, normalmente está visible en su página web
- Eres socia pero no tienes cuenta de la app: Si ya eres socia oficial del mercado, te debería haber llegado un correo para
registrar tu usuario en la esta app, revisa la bandeja de correo incluída la de spam, si no lo encuentras, ponte en contacto
con la persona administradora del mercado para que lo vuelva a enviar o te ayude con tu usuario
- No recuerdas la contraseña: Para esto, usa la opción "¿Has olvidado tu contraseña?". Te llegará un correo con instrucciones para 
reestablecerla
///

Una vez hemos entrado a nuestra área privada, tendremos una interfaz diferente dependiendo de si somos consumidora o 
entidad, como se detalla en las siguientes secciones. Pero hay un par de elementos comunes:


- Dos nuevas opciones en el menú de perfil (arriba a la derecha) para Editar perfil y Cerrar sesión
