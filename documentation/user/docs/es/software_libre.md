# :material-open-source-initiative: Software libre

Todo el trabajo desarrollado para esta página web y las aplicaciones móviles de Mercado Social está liberado bajo 
licencias de software libre. El movimiento de software libre busca la libertad de las personas para usar, estudiar, 
distribuir y mejorar los programas libremente mediante la liberación del código fuente de los mismos, fomentando 
la cooperación, transparencia y los bienes comunes.
