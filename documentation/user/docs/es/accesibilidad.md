# :material-human: Accesibilidad

En el diseño de las aplicaciones se tiene en cuenta la **accesibilidad** para hacer su manejo inclusivo para personas con 
discapacidad.

Se tienen en cuenta los siguientes elementos:

- Contraste de colores que cumplan al menos el nivel AA de la [WCAG](https://www.w3.org/WAI/standards-guidelines/wcag/). En la mayor parte de los casos, se cumple el nivel AAA.
- Tamaños de texto y adaptación para configuraciones de texto grande
- Adaptación de contenidos y navegación a los servicios de accesibilidad nativos de Android e iOS, sobre todo para 
lectura de pantalla

/// admonition | Mejora continua
    type: info
La mejora de la accesibilidad en las aplicaciones es un proceso y siempre se puede seguir mejorando. Tenemos el compromiso
de seguir adaptando y comprobando que las aplicaciones sean cada vez más accesibles.
