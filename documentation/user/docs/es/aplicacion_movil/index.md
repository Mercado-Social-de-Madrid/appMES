# :material-cellphone: Aplicación móvil

Existen aplicaciones móviles para las dos plataformas principales: Android e iOS. Estos son los enlaces de descarga:

- [App Store](https://apps.apple.com/es/app/mercado-social/id1458549528?platform=iphone) (iPhone/iPad)
- [Google Play](https://play.google.com/store/apps/details?id=net.mercadosocial.moneda){ target=blank } (Android)
- [Formato APK](https://github.com/Mercado-Social-de-Madrid/AppMES-Android/raw/refs/heads/master/app/release/app-release.apk){ target=blank }
(Archivo instalable para teléfonos Android. Esta opción está pensada para usuarias que no tengan 
los servicios de Google en su teléfono)

![Entidades app móvil](../../assets/entidades-lista.png){ loading=lazy align=left width=300px }

El objetivo de la app móvil es crear un vínculo más cercano de las usuarias con el MES, mejorando su experiencia y aprovechando
las ventajas de utilizar una aplicación móvil nativa con las siguientes características: 

- Información disponible sin conexión
- [Accesible](../accesibilidad.md)
- Alto rendimiento para interactuar de forma ágil
- Notificaciones push para avisos instantáneos



/// admonition | Compatibilidad
    type: info
Para hacer la app más inclusiva con quienes intentan alargar al máximo la vida de su teléfono y evitar en lo posible la obsolescencia, 
tratamos de hacer que la app sea compatible con la versión más antigua posible de cada plataforma.
Ahora mismo es compatible para teléfonos a partir de estas versiones:

- Android 5 (Lollipop)
- iOS 15.6
///
