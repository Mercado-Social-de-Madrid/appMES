# :material-bell-outline: Notificaciones

La aplicación móvil está preparada para recibir dos tipos de avisos con notificación:

- Noticias: mensajes que se envían desde la organización de cada mercado territorial
- Ofertas: las envían las entidades para sus promociones

Se recomienda activar las notificaciones para estar al día de las novedades del mercado social. Para ello, la primera vez
que se abra la app, hay que permitir las notificaciones en la ventana que aparece.

También se pueden habilitar o deshabilitar manualmente desde los ajustes del teléfono:
`Ajustes -> Aplicaciones -> Mercado Social -> Notificaciones`
