# :material-card-account-details-outline: Carnet digital


Esta función estará disponible para los mercados que tengan registro de socias consumidoras. Con este carnet digital, las socias
podrán acceder a ventajas y descuentos que ofrezcan las entidades de la red, es una forma sencilla de verificar la condición
de socia del mercado social correspondiente.

![Carnet socia consumidora](../../assets/carnet-socia-consumidora.png){ loading=lazy align=left width=250px }

En este carnet se muestran los datos personales o de entidad, el número de socia y un código QR con el que se puede validar 
el estado de la socia (activa o no activa en el mercado si se ha dado de baja).


/// admonition | Alta en el mercado
    type: info
Para conseguir este carnet digital, primero hay que pasar el proceso formal de alta o acogida en el mercado correspondiente.
Cada territorio suele tener un formulario de solicitud en el que se inicia el proceso de alta. Una vez finalizado este proceso,
llegará un correo para crear un usuario de la aplicación y poder acceder con él a la app para mostrar el carnet y acceder a las
ventajas de socia.
///

/// admonition | ¿Puedo llevar mi carnet sin tener la app móvil?
    type: tip
¡Si! Si no quieres instalar la app en tu teléfono pero quieres llevar a mano tu carnet para acceder a tus ventajas de socia
puedes tenerlo en formato imagen o imprimirlo físicamente en tamaño "tarjeta". Para esto tienes que acceder a tu área privada
en la web como se explica en [esta sección](../panel_gestion/consumidoras/index.md).
///

### Comprobar estado de socia
Las entidades tendrán una función añadida para comprobar si una socia está activa en el mercado para aplicarle o no la ventaja
que ofrece. Tiene dos opciones, usar la cámara para escánear el código QR de la socia, o introducir el número de social manualmente.
Estos botones están en la parte superior derecha de la pantalla del carnet digital.

![Comprobar estado socia](../../assets/comprobar-estado-socia.png){ loading=lazy width=400px }
